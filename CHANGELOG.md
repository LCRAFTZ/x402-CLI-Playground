All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-11-06
### Added
- Full offline-first x402 CLI implementation with `serve`, `pay`, `inspect`, and utility commands.
- Mock Facilitator with in-memory settlement and transaction persistence.
- Docker and CI/CD support with network-isolated testing.
- Strict `--json` output for automation.
